# Heimdall Examples

This folder contains rendered example documents for concept drift detectors implemented in `heimdall`.

- [ADWIN](./dfr_adwin.md): Adaptive Windowing applied to a numeric stream to illustrate distribution monitoring and virtual concept drift detection.
- [AEDD](./dfr_aedd.md): Autoencoder-based unsupervised monitoring of multivariate data for virtual concept drift detection.
- [KLDIST](./dfr_kldist.md): Kullback-Leibler divergence between reference and recent windows for virtual concept drift detection.
- [KSWIN](./dfr_kswin.md): Kolmogorov-Smirnov windowing over a numeric stream for virtual concept drift detection.
- [MCDD](./dfr_mcdd.md): Window-based mean comparison using statistical tests to identify virtual concept drift.
- [Page-Hinkley](./dfr_page_hinkley.md): Sequential change-point monitoring example over a numeric stream.
