# Heimdall Examples

This folder contains rendered example documents for concept drift detectors implemented in `heimdall`. The Markdown examples are available under [`examples/`](./examples/), with companion Word files under [`examples/doc/`](./examples/doc/) and extracted R scripts under [`examples/r/`](./examples/r/).

- [ADWIN](./examples/dfr_adwin.md): Adaptive Windowing over a numeric stream, illustrated as a virtual concept drift detector.
- [AEDD](./examples/dfr_aedd.md): Autoencoder-based unsupervised monitoring of multivariate data for virtual concept drift detection.
- [CUSUM](./examples/dfr_cusum.md): Sequential cumulative monitoring of a binary error stream for real concept drift detection.
- [DDM](./examples/dfr_ddm.md): Drift Detection Method applied to a binary error stream for real concept drift detection.
- [ECDD](./examples/dfr_ecdd.md): EWMA-based monitoring of classification errors for real concept drift detection.
- [EDDM](./examples/dfr_eddm.md): Early Drift Detection Method using distances between classification errors to detect real concept drift.
- [HDDM](./examples/dfr_hddm.md): Hoeffding-bound monitoring of the error stream for real concept drift detection.
- [KLDIST](./examples/dfr_kldist.md): Kullback-Leibler divergence between reference and recent windows for virtual concept drift detection.
- [KSWIN](./examples/dfr_kswin.md): Kolmogorov-Smirnov windowing over a numeric stream for virtual concept drift detection.
- [LBDD](./examples/dfr_lbdd.md): Levene-test-based comparison of window variability for virtual concept drift detection.
- [MCDD](./examples/dfr_mcdd.md): Window-based mean comparison using statistical tests to identify virtual concept drift.
- [Page-Hinkley](./examples/dfr_page_hinkley.md): Sequential change-point monitoring over a numeric stream, illustrated here for virtual concept drift detection.
